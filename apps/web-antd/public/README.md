# 应用运行时配置说明

## app-config.json

此文件包含应用的运行时配置，可以在应用打包后直接修改，无需重新编译。

### 配置项说明

- `title`: 站点名称，显示在左上角logo旁边和浏览器标签页标题中

### 使用方法

1. **开发环境**: 直接修改 `public/app-config.json` 文件
2. **生产环境**: 修改构建后的 `dist/app-config.json` 文件，然后刷新浏览器即可生效

### 示例

```json
{
  "title": "TTShang"
}
```

修改 `title` 的值后保存文件，刷新浏览器即可看到新的站点名称。
